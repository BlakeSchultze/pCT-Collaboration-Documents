	            \item \coloredtext*{documentation-fileset}{\textit{$<$Phantom$>$\_XXXX[\_AAA]\_xxx.dat.root.reco.root.bin}} : processed experimental data with tracker coordinates, recovery of missing hits when possible, and calibrated WEPL measurements for the object named \coloredtext*{documentation-subcategory}{``\textit{$<$Phantom$>$}''}, from run \# \coloredtext*{documentation-subcategory}{``\textit{XXXX[\_AAA]}''}, where \coloredtext*{documentation-subcategory}{``\textit{XXXX}''} is a 4 digit \# with leading zeros and \coloredtext*{documentation-subcategory}{``\textit{\_AAA}''} is an optional descriptor tag, and \coloredtext*{documentation-subcategory}{``\textit{xxx}''} is the gantry angle at which the data was acquired.
	        \end{deepList}
	    \end{deepList}
	\end{deepList}
